# SHORTFACTORY

**Decentralised creative economy. One human + Claude AI.**

---

### What is this?

ShortFactory is a self-sustaining content ecosystem where people make, play, rate, and distribute entertainment — no corporations extracting value in the middle. AI gates the door. Games hook engagement. Tools turn players into creators. Tokens give creative output real distributable value.

**Live now:** [shortfactory.shop](https://www.shortfactory.shop)
